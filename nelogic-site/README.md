## 🚀 About The Project

NeoLogic is a modern software studio focused on building scalable digital solutions for businesses.

This project represents the official NeoLogic landing page, developed with a modern UI/UX approach, smooth animations, responsive layouts, and a strong focus on performance and visual identity.

The website showcases:

- Web development services
- ERP systems
- REST APIs
- Business automation
- Founder presentation
- Success cases
- Contact system integrated with WhatsApp

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/site-NeoLogic.git
```

Access the folder:

```bash
cd site-NeoLogic
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
